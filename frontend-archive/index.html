Fledermaus-Tracking Analyse

Analyse von IR- oder Wärmebildvideos zur Fledermaus-Erkennung

Willkommen zum Fledermaus-Tracking Tool

Dieses Tool ermöglicht die Analyse von Videos zur Erkennung und Verfolgung von Fledermäusen in urbanen Quartieren.

Funktionen:

  • Video-Upload und Analyse
  • Echtzeit-Erkennung von Fledermausbewegungen
  • Statistische Auswertung der Flugbahnen
  • Visualisierung der Ergebnisse in Diagrammen
  • Quartierpotenzial-Analyse

Schnellstart:

  1. 1. Video hochladen (Abschnitt "Video Hochladen")
  2. 2. Analyse starten
  3. 3. Ergebnisse im Abschnitt "Ergebnisse" ansehen
  4. 4. Quartierpotenzial auf der Karte analysieren
